## Get for yourself

-   Setup the configs in `/src/data/config.js`

```
URL = "snipc.me";
EMAIL = `mail@snipc.me`;
GIT_URL = "https://github.com/realsnipc/snipc-website";
```

-   Hit `npm i` to install dependencies with npm

```
npm install
```


-   Customize and you are good to go!
